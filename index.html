HTML Forms Cheat Sheet

Form References

Form Elements

Tag Name Description
<textarea> Text Area Element Used when creating a text-box for a multi-line input
<form> Form Element Used to collect and send information to an external source
<input> Input Element Used to render a variety of input fields on a webpage
<select> Select Element Used to create a dropdown list
<datalist> Datalist Element Paired with input to create a search/autocomplete function
<label> Label Element Provides identification for a specific input
For additional information, check out this page

Form Types

Type Name Description
<input>: Checkbox Checkbox Type Renders a single checkbox item
<input>: Number Number Type Allows user to enter only numbers and a few special characters
<input>: Range Range Type/td> Acts as a selector between a minimum and maximum value
<input>: Text Text Type Renders a single row that users can type text inside
<input>: Radio Button Radio Button Type Renders a single radio button
<input>: Password Pasword Type Allows the user to type in censored text inside field
For additional information, check out this page

Form Attributes

Attribute Name Description
name Name Attribute Used as a key when form sends data
required Required Attribute Specifies the field must include a value
max Max Attribute Specifies maximum value of the input field
maxlength Max Length Attribute Specifies maximum number of characters that can be entered
min Minimum Attribute Specifies minimum value of the input field
minlength Min Length Attribute Specifies minimum number of characters that can be entered
